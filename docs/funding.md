# Funding & Support

Helix-Signal is **MIT-licensed open-source software** — a public good for stablecoin risk intelligence.

## Project Stance

- **No tokens.** No token launch, no ICO, no airdrop.
- **No VC.** Self-funded, community-supported infrastructure.
- **No closed-source monetization.** All features are free and self-hostable.

## How to Support

- **Use the platform** — deploy via `docker compose up -d`
- **Report issues** — open a GitHub issue for bugs or feature requests
- **Contribute** — PRs welcome, especially for new data sources, charts, and documentation
- **Spread the word** — tell teams who monitor stablecoin risk about Helix-Signal

## Infrastructure Costs

The public demo at [helix.withkeshav.com](https://helix.withkeshav.com) runs on a single VPS (~$15/mo) with:

- TimesFM forecasting model (3GB RAM)
- 7 RSS news feeds (free, unlimited)
- DefiLlama + DexScreener + CoinGecko APIs (all free)
- CryptoCurrency.cv news API (free, no auth)

Zero paid API keys required for core functionality.
